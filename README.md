# FSManagementExtended

FSManagementExtended - набор дополнительных типов объектов для [FSManagement](https://github.com/Vladislav117/FSManagement).

## Добавление зависимости

Для добавления зависимости необходимо внести изменения в `build.gradle`:

```groovy
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.Vladislav117:FSManagementExtended:0.0.1'
}
```

## Сборка

Сборка осуществляется командой `./gradlew build`

Результат сборки располагается в `build/libs`

## Документация

Документация представлена в исходном коде (javadoc).
